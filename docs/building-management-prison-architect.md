title: "Building management - Prison Architect"
type: idea
status: new
created: 2026-04-03T16:42:15
---

Instead of a god mode building system, that have a virtual store of objects and can place, destroy, move without having
to care about physical space, we can use a "realistic" building system, that we need some physical space to store the
objects not in use at the moment and move them around when needed.

Stacking/storage mode and stacking options should be available:
- Tables in storage mode should be upright, being able to hold multiple tables in a single tile
- Floor tiles should also be able to be stacked in storage mode

For this system to work better, we can consider having workers that move objects around (could be the coaches and
other gym personnel).
