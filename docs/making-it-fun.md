# Starting

First day is just for construction. In the second day it will be accepting new players.

# Day to day:

- Players arrive and practice
- You setup their training schedule, each day is one specific training (future: maybe more trainings during the day).
- You can setup a 1-on-1 training and decide the pairs: players learn more if paired with others with similar level or
better (you have coaches as well that can make others practice).

(Making it fun) Maybe the day is going to be too slow since we don't keep building the whole time.
We can have an "End day" button, like Mewgenics, or a "Fast forward" (maybe until next action required).

# Calendar

- 7 days a week, 4 weeks a month
- League every weekend
- Local championship every month
- Regional championship every 2 months
